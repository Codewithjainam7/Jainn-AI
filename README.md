## ğŸš€ **Quick Start**

### **Prerequisites**

```bash
- Node.js 18+ 
- npm or yarn
- API Keys (Gemini, OpenRouter, Supabase)
```

### **Installation**

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/jainn-ai.git
cd jainn-ai

# 2. Install dependencies
npm install

# 3. Set up environment variables
cp .env.example .env.local
```

### **Configure `.env.local`**

```env
# Gemini API (Required for main chat & image generation)
VITE_GEMINI_API_KEY=your_gemini_api_key_here

# OpenRouter API (Required for LLaMA & Mistral)
VITE_OPENROUTER_API_KEY=your_openrouter_api_key_here

# Supabase (Optional - for auth & chat history)
VITE_SUPABASE_URL=your_supabase_url_here
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key_here
```

### **Get Your API Keys**

<details>
<summary>ğŸ”‘ <b>Gemini API Key</b></summary>

1. Go to [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Click "Create API Key"
3. Copy and paste into `.env.local`

</details>

<details>
<summary>ğŸ”‘ <b>OpenRouter API Key</b></summary>

1. Visit [OpenRouter.ai](https://openrouter.ai/)
2. Sign up and go to "Keys"
3. Generate a new API key
4. Add credits to your account

</details>

<details>
<summary>ğŸ”‘ <b>Supabase Setup</b> (Optional)</summary>

1. Create a project at [supabase.com](https://supabase.com)
2. Go to Project Settings â†’ API
3. Copy the URL and anon public key
4. Run the SQL in `supabase_chat_sessions.sql` to create tables

</details>

### **Run the App**

```bash
npm run dev
```
